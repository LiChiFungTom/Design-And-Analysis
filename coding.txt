1. what is the GCD(original) python code and Pseudo-code?
    

    Python code:
    def gcd(a,b):
        while a!=b:
            if a > b:
                a = a - b
            else:
                b= b - a
        return a

    Pseudo-code:
        gcd(a,b):
            while a =/ b:
                if a > b:
                    a = a - b
                else:
                    b = b - a
            return a

    def gcd2(a,b):

            a,b = b,a
